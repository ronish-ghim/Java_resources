## Conclusions

**Lab 1:**

This lab demonstrated event handling using mouse actions in Java Swing. It helped in understanding how different mouse events (press and release) can trigger different operations, improving knowledge of interactive GUI behavior.

**Lab 2:**

This lab provided practical experience with Swing components and action events. It reinforced the concept of handling button clicks to perform arithmetic operations and display results dynamically.

**Lab 3:**

This lab enhanced understanding of GUI-based calculators with multiple operations. It strengthened knowledge of event-driven programming and user input handling in Swing applications.

**Lab 4:**

This lab introduced adapter classes for handling events efficiently. It simplified event handling implementation and improved understanding of reusable event-handling techniques.

**Lab 5:**

This lab demonstrated the use of GridLayout for designing structured forms. It helped in understanding component alignment and layout management for simple login interfaces.

**Lab 6:**

This lab provided deeper insight into GridBagLayout, a flexible and powerful layout manager. It improved skills in designing complex and responsive GUI forms.

**Lab 7:**

This lab focused on BorderLayout and panel organization. It enhanced understanding of dividing a frame into regions and combining multiple layouts for building complex interfaces.

**Lab 8:**

This lab demonstrated handling button events and dynamically changing component properties like color. It improved understanding of user interaction and visual feedback in GUI.

**Lab 9:**

This lab combined multiple layout managers in a single application. It strengthened knowledge of structuring complex GUIs and handling multiple events effectively.

**Lab 10:**

This lab introduced basic graphics in Swing. It helped in understanding how to draw shapes and respond to user actions to render graphical content.

**Lab 11:**

This lab demonstrated the use of radio buttons and ItemListener. It improved understanding of selection-based input and dynamic UI updates.

**Lab 12:**

This lab focused on toolbars and icons in Swing. It enhanced knowledge of building professional-looking interfaces and handling toolbar actions.

**Lab 13:**

This lab introduced JDBC for database interaction. It helped in understanding how to insert and retrieve records and sort data using SQL queries.

**Lab 14:**

This lab demonstrated updating database records using JDBC. It strengthened understanding of executing update queries and applying conditions.

**Lab 15:**

This lab provided knowledge of database connectivity and prepared statements. It emphasized secure data insertion and prevention of SQL injection.

**Lab 16:**

This lab demonstrated navigation through ResultSet. It improved understanding of cursor movement methods like next(), previous(), first(), and last().

**Lab 17:**

This lab introduced updatable ResultSet. It enhanced understanding of retrieving, modifying, and updating records dynamically in a database.

**Lab 18:**

This lab explained the servlet life cycle. It helped in understanding initialization, service, and destruction phases of a servlet.

**Lab 19:**

This lab demonstrated session management in servlets. It improved understanding of storing and retrieving data across multiple requests.

**Lab 20:**

This lab introduced cookies in servlets. It helped in understanding client-side data storage and retrieval.

**Lab 21:**

This lab extended cookie handling using JSP. It improved understanding of integrating JSP with client-side storage mechanisms.

**Lab 22:**

This lab demonstrated form handling using JSP. It helped in understanding request forwarding and data retrieval between pages.

**Lab 23:**

This lab implemented a simple interest calculator using JSP. It strengthened knowledge of form processing and dynamic content generation.

**Lab 24:**

This lab introduced Remote Method Invocation (RMI). It helped in understanding distributed computing and communication between remote Java objects.
